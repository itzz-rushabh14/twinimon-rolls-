How to Setup Card
=============
First things first. Your gonna start by downloading the template-code.html file in this repository.
After you're done with that. Simply open the code in a Text Editor or your notepad or anything you can edit it with.
Simply replace ALL of the Text and Areas with "Your name" Etc. 

Requesting Themes
=============
You can request a new theme if you dont like the default water theme, Just put in a ticket in our Discord.
ONLY GitHub Access Users can have cards. We do not accept anybody.

If you need assistance you can DM Me on Discord at YoungDollazz1700 or Ping @web development roles in our discord.

What not to do
=============
Do not mess with the actual code. Dont remove any "" in the code unless you add them. 
When break texting like going down a line
like this in your bio use two br codes exactly like this >     Wit this <br><br> i am broken text

How to Submit the Card
=============
Upload it to our website in a folder. Or have a Web Dev Help guide you through how to make your first time folder.
and how to upload the code. For changing any links or titles you can do it within the GitHub app or site.
